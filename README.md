## 📦 Instalación

1. Clonar el repositorio

```bash
git clone https://github.com/carlosdm121/cuestionariofcc.git
```

2. Entrar a la carpeta

```
cd cuestionariofcc
```

3. Abrir el archivo

```
index.html
```
